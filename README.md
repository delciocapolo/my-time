# Descrição

**My Time** é uma aplicação minimalista e produtiva, desenvolvida com Tauri, que ajuda-nos a manter o foco e gerir melhor o nosso tempo. Inspirado na técnica Pomodoro, o My Time divide o nosso trabalho em ciclos equilibrados de foco e descanso, permitindo-nos alcançar mais resultados com menos stress.

Com uma interface simples, leve e rápida, o My Time adapta-se ao nosso fluxo diário, oferecendo ferramentas para aumentar a concentração, acompanhar o nosso progresso e criar hábitos mais saudáveis de produtividade.

**Principais destaques:**

- Sessões de foco e pausas totalmente configuráveis
- Interface limpa e agradável, pensada para não distrair
- Notificações inteligentes de início e fim de ciclos
- Histórico para acompanhar a consistência
- Aplicação leve e segura graças à estrutura Tauri

**My Time — o teu espaço de foco, organização e evolução.**
